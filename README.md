###  Featured Project

####  Blockchain Voting System (BVS)
> A decentralized, tamper-proof voting system built on Ethereum Sepolia Testnet

-  Smart Contract — Solidity (vote, addCandidate, toggleElection)
-  Frontend — React.js + Tailwind CSS deployed on Vercel
-  Backend — Node.js + Express + MongoDB Atlas on Render
-  Web3 — MetaMask + Ethers.js + Infura RPC
-  Security — Double voting prevention, session management, time-bound voting

 [Live Demo](https://blockchain-voting-app-4z91.vercel.app) | [GitHub](https://github.com/Jamshed-Alam01/blockchain-voting-app)
